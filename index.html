Sheet of Paper

Sheet of Paper (SoP) is a small library to create and print text documents using HTML language.

We provide you several default classes to organize your document. For example, you can have a header, a body and a footer. Also, you can divide your header and footer into 3 different sections, by using .sop-left, .sop-right and .sop-center classes.

Default styles have been also set for titles and text, so you can use:

Paragraph, bold, italic and underlined.

You also can center text.

Or align it to the right.

Or even justify it, in order to have a more elegant document to present whoever you want, and margins will be similar and very straight on both sides of the paper. This is a very cool feature that we can use whenever we want.
